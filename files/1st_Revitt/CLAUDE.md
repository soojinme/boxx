## UX 핵심 결정
- 피드 반응: 나도! / 다녀왔어요 / 가고싶다
- 리뷰 평가: 🤫나만알고싶어 / 🙋친구한테만 / 📢다알았으면 / 🔥온세상이
- **리뷰 공유욕구 기본값: 🤫 나만알고싶어** (진입장벽 최소화)
- 리뷰 작성 모드: **초간단(사진+공유욕구만)** / 상세(전체 입력)
- 피드: 팔로잉 역순 시간순 (알고리즘 없음)
- 무드 배너: 실시간 집단 감성 한 줄 표시
- **최우선 과제: 구경꾼(서희형)을 자연스럽게 올리게 만들기**
