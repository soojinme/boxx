# Revitt — 핵심 유저 스토리

> 형식: `[페르소나]는 [목적]을 위해 [기능]을 원한다`
> 페르소나: 소연(올리는 사람) / 준호(깊은 사용자) / 서희(구경꾼 → 전환 타겟)
> 최종 업데이트: 2026-03-17

---

## 1. 온보딩 / 로그인

| # | 유저 스토리 | 우선순위 |
|---|---|---|
| US-01 | **서희**는 가입 장벽 없이 빠르게 앱에 들어오기 위해 **카카오 소셜 로그인 1탭**을 원한다 | 🔴 필수 |
| US-02 | **소연**은 내가 좋아하는 것만 피드에 보이게 하기 위해 **관심 카테고리(음식/영화/도서) 선택**을 원한다 | 🔴 필수 |
| US-03 | **서희**는 지역 기반 커뮤니티에 속한 느낌을 받기 위해 **내 동네 설정**을 원한다 | 🟡 중요 |

## 2. 홈 피드

| # | 유저 스토리 | 우선순위 |
|---|---|---|
| US-04 | **서희**는 지금 주변 사람들이 뭘 하는지 보기 위해 **팔로잉 최신순 피드**를 원한다 | 🔴 필수 |
| US-05 | **서희**는 공감을 표현하고 싶지만 리뷰 쓰기는 부담스러워서 **나도! / 다녀왔어요 / 가고싶다 반응 버튼**을 원한다 | 🔴 필수 |

## 3. 리뷰 작성

| # | 유저 스토리 | 우선순위 |
|---|---|---|
| US-09 | **서희**는 긴 글 없이 지금 이 순간을 기록하기 위해 **사진 1장 + 공유욕구 선택만으로 올리는 초간단 모드**를 원한다 | 🔴 필수 |
| US-10 | **준호**는 내 감상을 제대로 남기기 위해 **장소/제목 태그 + 텍스트 + 별점이 있는 상세 모드**를 원한다 | 🔴 필수 |
| US-11 | **소연**은 이 리뷰가 얼마나 퍼지길 원하는지 표현하기 위해 **공유욕구 4단계(🤫/🙋/📢/🔥) 선택**을 원한다 | 🔴 필수 |
| US-12 | **소연**은 장소를 직접 입력하지 않고 바로 찾기 위해 **카카오/네이버 맵 연동 장소 검색**을 원한다 | 🔴 필수 |
| US-13 | **준호**는 영화·도서도 쉽게 태그하기 위해 **TMDB / Google Books API 연동 검색**을 원한다 | 🔴 필수 |
| US-14 | **서희**는 실수로 올리는 걸 막기 위해 **올리기 전 미리보기 확인**을 원한다 | 🟡 중요 |

## 서희 전환 핵심 스토리 (최우선)

> 구경꾼 → 올리는 사람으로 전환하는 핵심 경로

```
1. 카카오 1탭 가입 (US-01)
       ↓
2. 피드 구경 중 반응 버튼 첫 클릭 (US-05)
       ↓
3. 초간단 모드로 사진 1장 올리기 (US-09)
       ↓
4. 공유욕구 🤫 선택 → 부담 없이 게시 (US-11)
       ↓
5. 내 리뷰에 반응 알림 수신 → 재방문 (US-26)
```
