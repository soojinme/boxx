# Revitt — 결정사항 로그

> 보스 + 오케스트레이터 확정 항목만 기록. 미결은 ❓표시.

---

## 서비스 기본
| 항목 | 결정 |
|---|---|
| 앱명 | **Revitt** (레빗) |
| 서비스 정체성 | 리뷰 기반 **소셜SNS** (리뷰는 수단) |
| 핵심 DNA | 실시간성 — "지금 이 순간 세상은 나 혼자가 아니다" |
| 설계 원칙 | 단순하게, 진입장벽 낮게 |

## 1단계 범위
| 항목 | 결정 |
|---|---|
| 시작 카테고리 | **v1: 카테고리 없음 (통합 피드)** → v2~3에서 음식/영화/도서 추가 |
| 음식 태그 DB | v2 이후 (카카오맵 + 네이버맵 API) |
| 영화 DB | v2 이후 (TMDB API) |
| 도서 DB | v2 이후 (Google Books API) |
| 네비게이션 | 홈 / 둘러보기 / 리뷰찍기(중앙) / 검색 / 마이페이지 |
| 전체 피드 알고리즘 | **최신순** |
| **첫 런칭 플랫폼** | **iOS 네이티브 (Swift/SwiftUI)** |
| 이후 확장 | Android |

## 브랜드 아이덴티티
| 항목 | 결정 |
|---|---|
| 브랜드 메인 컬러 | **오렌지** |
| 컬러 시스템 | 시간 기반 다이나믹 테마 — 오전(라임그린) / 오후(오렌지) / 저녁(바이올렛) |
| 캐릭터 | 토끼 (Revitt = Review + Rabbit) |
| 모드 | 라이트 모드 우선 · 다크모드는 Figma 변수(Variable)로 추후 추가 |
| 폰트 | Pretendard |
| 모서리 | 라운드 (pill/bubble 스타일) |
| 마케팅 채널 | 인스타 > TikTok > X (계획만, 실행 추후) |

## UX / 인터랙션
| 항목 | 결정 |
|---|---|
| 피드 반응 | **나도! 하나만** — "나도 해봤어 / 나도 가고싶어" 통합 의미. 재탭 시 취소 가능 |
| 피드 방식 | 팔로잉 역순 시간순 (알고리즘 없음) |
| 홈 피드 무드 배너 | 실시간 집단 감성 한 줄 표시 (예: "🌧️ 비 오는 밤, 다들 라멘 먹는 중") |
| 출시 전략 | Waitlist → **지역 기반 순차 오픈** (관심사 태그 선택 포함) |
| 첫 오픈 지역 | 서울 마포·홍대·이태원 → 서울 전체 → 수도권 → 전국 |
| 리뷰 평가 시스템 | **공유 욕구 4단계** (🤫나만알고싶어 / 🙋친구한테만 / 📢다알았으면 / 🔥온세상이) |

## 온보딩 플로우
| 스텝 | 내용 |
|---|---|
| 1 | 소셜 로그인 (카카오 / Apple / Google) |
| 2 | 닉네임 설정 |
| 3 | 완료 화면 → 피드로 이동 |
> 지역 설정은 마이페이지 > 설정에서 나중에 변경 가능

## 핵심 설계 과제
| 항목 | 내용 |
|---|---|
| **최우선 과제** | 구경꾼(서희형)을 자연스럽게 올리는 사람으로 전환 |
| 접근 방향 | 공유욕구 4단계로 진입장벽 낮추기 + 리뷰 올리기 마찰 최소화 |
| **리뷰 작성 모드** | 초간단 모드: 사진 1장 + 공유욕구 선택 → 즉시 올리기 (장소·텍스트 선택사항) |
| **사진 필수 여부** | 선택사항 — 사진 없으면 **감성 텍스트 카드 자동 생성** (배경색 + 공유욕구 이모지 + 텍스트) |

## 공유욕구 공개 범위 정책
| 단계 | 공개 범위 | 피드 | 둘러보기 | 검색 |
|---|---|---|---|---|
| 🤫 나만알고싶어 | **본인만** | ✕ | ✕ | ✕ |
| 🙋 친구한테만 | **팔로잉에게만** | ✓ | ✕ | ✕ |
| 📢 다알았으면 | **전체 공개** | ✓ | ✓ | ✓ |
| 🔥 온세상이 | **전체 공개 + 추천 강화** | ✓ | ✓ (상단) | ✓ |
